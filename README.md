# Pathfinder App

## Project Structure

This project is structured to support a MERN stack application. It contains separate directories for the client and server, along with shared configuration files. Below is a brief overview:

### Directory Structure

```
pathfinder-app/
├── client/            # React frontend
│   ├── package.json   # Client dependencies
│   └── ...           # Other frontend files
├── server/            # Node.js/Express backend
│   ├── package.json   # Server dependencies
│   └── ...           # Other backend files
└── README.md          # Project overview
```

### Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/swara777/pathfinder-app.git
   cd pathfinder-app
   ```

2. **Install dependencies**:
   - For the client:
     ```bash
     cd client
     npm install
     ```
   - For the server:
     ```bash
     cd server
     npm install
     ```

3. **Run the applications**:
   - Start the client:
     ```bash
     cd client
     npm start
     ```
   - Start the server:
     ```bash
     cd server
     npm start
     ```
